# HOW TO USE

[Download .NET 8 Runtime for both systems](https://dotnet.microsoft.com/en-us/download/dotnet/8.0)


# Server Side

[Install VB Audio Cable](https://vb-audio.com/Cable/index.htm)
       
#       Console Version
for the server side cd to the folder and run this

       MicServer.exe 9000 "CABLE"

# UI Server Version

Click output device and click the VB Audio Cable then click start


<img width="405" height="297" alt="image" src="https://github.com/user-attachments/assets/7b59f720-8b08-497f-8710-7a6224c21929" />



# Console Client Side 
  cd to the folder in CMD and run this command
  
      MicClient.exe YOUR_LOCAL_IP_TO_THE_MACHINE 9000

# UI Client Side


Select your microphone make sure the port is the same on both machines type in your local IP and click start
